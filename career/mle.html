Data Scientist - Machine Learing Engineering

Responsibilities

  • Be part of a team of innovative machine learning experts and growing team's MLOps capacity
  • Define business problems as mathematical models that ML models/algorithms can solve, train & evaluate ML algorithm/models, and perform model montoring and maintenance.
  • Deploy model into production, and set up data & model pipelines for production deployment
  • Understand end user usage, translate business needs into an AI/ML app, and use appropriate tools & approach for the app

Qualifications

  • Solid background in developing ML models/algorithms to solve business problems
  • Past experience in ML development tools in python (e.g. flask/django, skearn, pytorch, xgboost/lightgbm)
  • Strong coding skills in python (with Proficiency in e.g. numpy, scipy, pandas, etc,) or R (we can chat about that) and writing production-ready code
  • Past experience conducting code review and/or managing codebase
  • (Nice to Have) Past experience working with containers and pipelines (e.g. kubernetes, docker, airflow)
  • (Nice to have) Past experience mentoring non-MLE data scientist and entry level engineer
